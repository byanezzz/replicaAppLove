# AppLove

## Contenido general del repositorio

- Encontrarás un archivo base `index.html` el cual contiene la estructura del proyecto.

- Una carpeta `css` donde tendrás un archivo base `main.css` el cual contiene clases reutilizables, y clases específicas para ciertas secciones.

- Además una carpeta `assets` donde se encuentra la carpeta `images` en la cual están alojadas todas las imágenes usadas en el proyecto.

## Estructura del proyecto
- Se tiene un `header` estático, en donde se muestra el menú de navegación.
- Seguidamente hay una sección de presentación o home donde se se da una pequeña idea de lo que trata el negocio.
- Posterior a esto, se tiene otra sección donde se detallan los servicios que se prestan.
- Luego, se muestra un pequeño portafolio de trabajos realizados anteriormente. 
- Por último, un apartado o `footer` donde se colocan los derechos de autor.

## Estilos del proyecto
- Por un lado, se cuenta con clases genéricas las cuales permiten dar el mismo estilo a diferentes partes del proyecto: 
    - .mb-0: Quitar margen inferior
    - .mt-0: Quitar margen superior 
    - .m-0: Quitar todos los márgenes
    - .text-uppercase: Transformar textos a mayúsculas
    - .inline-block: Convertir elementos a inline-block

- De igual forma se tienen clases que dan estilos específicos a secciones y/o elementos del proyecto.
